Q3.Explain the use of grid-auto-row and grid-auto-column using code examples

Ans. The grid-auto-rows and grid-auto-columns properties in CSS Grid are used to define the size of rows and columns that are automatically created when grid items are placed outside the explicitly defined grid structure. For example, if you define only two columns using grid-template-columns and then add a third item, it will automatically be placed in a new row. In this case, grid-auto-rows controls the height of that new row. Similarly, if you only define rows and an item is placed in a column that wasnâ€™t explicitly defined, then grid-auto-columns determines the width of that new column. These properties are helpful for managing the layout of dynamically added or positioned grid items, ensuring that even implicitly created rows or columns maintain consistent sizing
